## 缠论里的定理
**场中说禅走势分解定理一**：任何级别的走势，都可以分解成*同级别*「盘整」，「下跌」与「上涨」三种走势类型的连接。
>注解：同级别，要求笔跟笔，线段跟线段，不可混为一谈
走势分来分去就这三种，再无其他

**场中说禅走势分解定理二**：任何级别的任何走势类型，都至少由三段以上次级别走势类型构成
>两个关键点
>* 至少三段
>* 次级别走势类型

**第三类买卖点定理**：一个次级别走势类型向上离开缠中说禅走势中枢，然后以一个次级别走势类型回试，其低点不跌破ZG，则构成第三类买点；一个次级别走势类型向下离开缠中说禅走势中枢，然后以一个次级别走势类型回抽，其高点不升破ZD，则构成第三类卖点。

## 买卖点的总结

用缠论赚了我以前从来不敢想的钱，市场真的就变成了提款机，当然也可以提我的款，一波赚了7000，大意亏回去4000，还好提前变现了3000，落袋为安，真的是凭运气赚得钱，凭实力亏回去，所以为了让德配上财，继续学习和提高认知。

缠论最核心的就是**买点买，然后卖点卖**，操作就这两个，别无其他，难点在于思考哪里是买点和卖点，对市场的走势的思考和分析，工作都在脑袋里，这也算不上什么秘密，只是别人看不上，看不起，看不懂，以为价格分析都是神棍，但陶冶了1年多，经历另一个牛熊，终于还是转变了观念，开始了学习，首先感谢自己，总是肯付费折腾，花钱报班没停过，遇上贵人也是必然的吧，扭转了我对K线技术的观念，让我知道了缠论。

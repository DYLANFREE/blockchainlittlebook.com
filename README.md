### 点线面体
在了解K面和K体前，我们先来回顾一下点线面体，接触过义务教育的，都在初高中学习过几何，几何把物体分成点线面体，一条线是由无数密集点组成的，一个面是由无数线密集排列而成，一个体是由无数面密集排列而成。

![6276aV.png](https://s3.ax1x.com/2021/03/18/6276aV.png)](https://imgtu.com/i/6276aV)

类比到市场里，K线K面K体也是顺利成章的，K柱组合成K线，K线组合成K面，K面组合成K体。市场走势图里最小的组成元素点，理应是一笔笔的交易，毕竟市场就是由一笔笔交易构成而成的，一笔交易包括三个信息：
o	时间
o	价格
o	数量

但一笔笔的交易就像化学里的原子，细致细微但不方便分析，而原子的集合体，元素不如原子细微，却可以集中体现某一类原子的性质。

把一笔交易比作原子，把一笔交易对应的元素便是K柱，K柱的方便之处在于，用图表的方式直观展现了一段时间内价格的运动，比如1分钟K柱指的是1分钟内价格的上下移动，也就是说不单单看一笔交易，而是看这一分钟内打包的所有交易体现的属性。

一个X分钟的K柱可以直观展示出，开盘价，收盘价，最高价，最低价，以及在X分钟时间内，价格是如何波动的。

![](https://i.loli.net/2021/03/18/T2NQK9nzEYXgoFS.png)

进一步，由Y个连续X分钟K柱组合成的是K线，一段Y * X分钟的K线可以直观展示价格在Y * X分钟内的移动的轨迹。

![](https://i.loli.net/2021/03/18/EcFsa6u43iKCNT1.png)
 
进一步，由Z个连续Y * X分钟K柱组合成的是K面，一段Z * Y * X的K面可以直观展示价格在Z * Y * X分钟内的价格移动轨迹。

![](https://i.loli.net/2021/03/18/gnR2m7wtNIqiZAJ.png)

这就是不同的之处了，一个时长24小时的K线图，你可以当成一段24小时的K线，也可以看成是每根K柱1小时，每根K线由4根1小时K柱组成，整个K面由6根4小时的K线组合而成，整个K面由24根1小时的K柱组合而成。

同理可得，由N个连续Z * Y * X分钟K柱组合成的是K体，一段N * Z * Y * X的K体可以直观展示价格在N * Z * Y * X分钟内的价格运动轨迹。

![](https://i.loli.net/2021/03/18/JfX6dtKVM5COEQm.png)

整一个柱线面体的升级用数学语言描述就是

设一个K柱为K1,则多个连续K柱的集合便是K线
｛K1,K2,K3,K4···Kn｝=X（X是K柱构成的线）
｛X1,X2,X3,X4···Xn｝=Y（Y是X线构成的面）
｛Y1,Y2,Y3,Y4···Yn｝=Z（Z是Y面构成的体）
······

### 为什么要对K线分类和组合呢？
因为不同的视角观察同一段走势可以得出不同的信息，就好比同样是一个正方体，如果让我们去观察正方体的性质，不同维度视角得出来的信息是不一样的，或者说信息的颗粒度是不一样的，细致到点的视角能得出微观的信息，整体到体甚至更高维度的视角能看出宏观的走向。

![](https://i.loli.net/2021/03/18/23rwoKgFCVh6YiI.png)

同理，如果说看K线是在一维观察价格走势，看K面就是在二维观察价格走势，看K体就是在三维观察价格走势，K面和K体以及更高维度的视角，是对同一个价格走势的深度信息挖掘，是对单一视角的降维打击。

![](https://i.loli.net/2021/03/18/2JskIFNnYAEWG6q.png)
 
还是上面三张图来做例子，帮助直观感受一下，K线的图，你看到的是乱序的，只能看到K线的价格的移动轨迹

![](https://i.loli.net/2021/03/18/EcFsa6u43iKCNT1.png)
 
K面的图，你看到的是3段K线的有序组合，看到的是一段下跌K线+一段横盘K线+一段上涨K线

![](https://i.loli.net/2021/03/18/EcFsa6u43iKCNT1.png)
 
K体的图，你不仅能看到每一个单独的K面的结构是如何的，第一个K面上涨，第二个K面下跌盘整，第三个K面上涨盘整，你还能看到由这三个K面组合成的整一个K体的结构是上涨+横盘。

![](https://i.loli.net/2021/03/18/gnR2m7wtNIqiZAJ.png)
 
你发现没有，仅仅是通过简单的分类，那些分类都是我随手花的框，只是为了让大家明白缠论眼里的K线结构，即使如此，同样的K线图，在你的眼里，便不再是原来的那个K线图，竟然能得出不一样的结论，你能看到一个K体，K体里的K面，K面里的K线，K线的K柱。恭喜你，K线还是那个K线，但你不再是那个你，如果你能理解上文所说的东西，你对K线的理解，起码已经进入第二层，看K线不再是K线。

### 上涨-横盘-下跌

先来解决第一个问题，你该采取什么样的方式来对走势进行分类呢？答案你不会陌生，就是「上涨-横盘-下跌」。

市场的走势可以用这么一个活成段子的真理来概括：不是上涨就是下跌，不排除有横盘的可能性。

![](https://i.loli.net/2021/03/18/o51yJQUdgeYLPnx.png)

这是市场里的一句“废话”，经常在群里讨论走势时出现，暗讽分析师之无用，是一个人人都知道的废话，但此“废话”之所废，是因为单单这么一句话，它无用，派不上用场，但不代表它不正确，相反，它太过于正确，是一个被人们习惯性忽略的公开秘密。这个公开的秘密放在群里是段子，放在缠论里便是公理，是缠论最底层的地基，一点点开枝散叶构建起缠论的整座大厦，仅是理解这个秘密，在宏观上，你的交易胜率就已经战胜市场里2/3的人，没有夸张，只是一个简单的数学统计和分类。

走势在A时刻，有且只有三种情况的其中一种情况，价格向上移动，价格保持不变，价格向下移动，价格向上移动，上涨，价格保持不变，盘整，价格向下移动，下跌，这是毋庸置疑的，你找不到第四种情况。
对市场做分类后，仅从数学角度上说，我们某一次买入，盈利概率便是50%，涨了你赚，横盘不亏，跌了你亏。在三种分类的基础上，对走势进行排列组合，走势组合不外乎以下六种情况，A时刻+B时刻+C时刻，假设你在A时刻买入，C时刻卖出，那么所有排列组合的结果如下表

| A时刻+B时刻+C时刻| 结果 |
| --- | --|
| 下跌+盘整+上涨 | 买入盈利 |
| 下跌+盘整+下跌 | 买入亏损 |
| 下跌+上涨     | 买入盈利 |
| 上涨+下跌     | 买入亏损 |
| 上涨+盘整+下跌 | 买入亏损 |
| 上涨+盘整+上涨 | 买入盈利 |

基于这个认知，你做交易的胜率就已经大于别人了，为什么？因为对于以下两个问题，你有了清晰的认识。
什么时候该买入？上涨的时候不能买；
什么时候该卖出？下跌的时候不能卖。

盘整相当于价格不动，等效于省略，在上涨中买入，一共有3种结果

上涨+盘整+上涨，盈利

上涨+盘整+下跌，亏损

（盘整+）上涨+下跌，亏损

所以在上涨买入，你可能会盈利，但从理论上计算，你盈利的概率是1/3。

上涨的时候不能买，自然要在下跌中买入

下跌+盘整+下跌，亏损

下跌+盘整+上涨，盈利

（盘整+）下跌+上涨，盈利

下跌中买入，你当然可能会亏损，但从理论上计算，你盈利的概率是2/3。

因此从这个角度说，在市场不追涨，宁愿错过也不要过错，坚持只在下跌中买入就可以让你的交易胜率达到66.66%，市场都忍不住给你喊666。

### 混沌市场里的简单秩序

在一个无限机会的市场里，试图用有限的本金去抓无限的机会，这本身就是超脱能力之外的贪欲，这贪欲便是从古至今财富转移的动力源泉，驱动着财富从低往高处流动，从认知低的人手里流动到认知高的人手里。从一个公开的秘密出发，我们深想多了一层，仅仅是想多了一层，便可以让我们赚钱的概率从50%蜕变到66.6%，当然这仅是理论知识上的计算，说是文字游戏，完全可以，理论具体到实际操作里情况会复杂

比如有人就是在上涨中买入，但他1/3的盈利总金额，大过他2/3的亏损总金额，他最终情况仍旧是盈利，简单的来说就是交易3次，赚1次赚了10万，亏2次每次亏了3万，最后仍旧是盈利4万。

比如有人就是在下跌中买入，即使他交易3次，理论盈利的次数是2次，但一次亏损>二次盈利+本金。

简单的理论和复杂的现实存在差异，但这不影响追涨杀跌理论的存在和正确，就像一个复杂的四则运算有加减乘除平方根号括号等等，但再怎么复杂的运算都是基于1+1=2演化出来的。总而言之，一个复杂的四则运算里，可能连加号都没有，你根本看不到1+1=2，但不影响1+1=2是所有四则运算的底层构建的事实，复杂的现实有无数的因素让下跌中买入的单次结果可能是亏钱的，但并不影响长期整体里，跌买的盈利概率是2/3。

知识和运用知识是两回事，市场没有那么理想，无法那么微观，不存在假设前提，但此时此刻，现实的复杂，不妨碍我们知道市场里存在这么一个本质的东西，不妨碍我们理解不追涨不杀跌理论背后存在的数学概率，这是市场里看不见的数学之手。

![missinghand](https://github.com/DYLANFREE/huangjun.work/blob/master/images/2class/missinghand.gif)

如果你能正确理解上面的东西，恭喜你，你已经在混沌的市场里找到了一个最简单的秩序。基于这个秩序，我们想象这么一个交易世界，每一次的交易，盈亏的金额是固定的，比如赚就赚50，亏也亏50，那么，只要交易次数足够多，大数定律，**终会让坚持在下跌中买入的人实现盈利，会让坚持在上涨中买入的人实现亏损。**

这有点类似抽卡游戏，程序写死了出货的概率，再怎么悲惨的非酋，只要抽的多，该中的迟早会中。

### 为什么市场里盈利的总是少数人？

这不是什么二八定律，二八定律是用表面现象总结表面结果，说个稀里糊涂，懂个懵懵懂懂。真正的原因是绝多大数人的绝大部分操作是追涨杀跌，短时间内有人可以被幸运女神眷顾，成为市场里赚钱的追涨幸存者，但长时间里“命运死神”会悄无声息地带走这些人。追涨韭菜的非酋结局是一开始注定的，像命中注定一样，他们会抱怨命运不公，会感叹时运不济，<u>但命运是公平的，公平在他的不公是他自己选择的</u>，他们眼里所谓的天生亏钱命，在我们眼里而言，不过是数学的因果律罢了。

### 所有交易理论的源头

从整个市场的宏观角度来说，对于一个人的长期多次交易而言，知道在下跌中买入，这么一个简单的常识便可以让交易盈利的概率达到2/3，但这够吗？对于一些人来说够了，对我们肯定远远不够，下跌中买入还会有1/3的亏损可能性，我们要的是100%胜率，即使做不到100%，也要尽可能靠近100%。

那怎么做到100%，思路很简单，下跌中买入的结果就三种
下跌+盘整+下跌，亏损

下跌+盘整+上涨，盈利

下跌+上涨，盈利

只要你能把第一种亏损的情况排除，不就是100%盈利了？这个理解用人话说，你肯定会理解成废话，甚至嗤之以鼻一笑了之，翻译过来就是——<u>只要跌完了不再跌，买入不就100%赚钱了？</u>

如果全文就这句话，“交易很简单，跌完了再买就可以啦”，换做我是读者，我都会顺着网线爬过去打我自己两巴掌，但显然你现在应该是巴不得给我点在看，甚至给我赞赏一下。收回来，整篇公理，推导，看下来，你和只懂那句话的人是完全不一样的，“跌完再买”是没有一点儿指导意义的，而基于理想市场的微观推导之后的完全组合分类是当下就可以起到作用的。

在某一次看盘里，你情绪一动想追涨，理智会立刻告诉你不行，即使你最后忍不住追了，亏了之后，你也是亏得明明白白的，除非你能把上面的所有东西都忘掉，不然你是回不到你那个过去的旧世界。
除了不要追涨，让你在思想上就已经领跑2/3的人，你还可以怎么样，你当下就可以设计一个100%赚钱的交易系统。

A时刻下跌介入后，如果B时刻不是上涨，无条件退出

下跌+盘整+下跌，下跌买入，盘整立刻退出，不赚不亏

下跌+盘整+上涨，下跌买入，盘整立刻退出，不赚不亏

下跌+上涨，下跌买入，上涨卖出，盈利

这就是一个理论上100%盈利交易体系，虽然实际中操作起来得考虑很多其他因素，但不影响你交易的思路。

## 缠论里的定理

中枢的魔力根源是什么？
缠中说禅：所有人的贪嗔痴

**缠中说禅盘整**：在任何级别的任何走势中，某完成的走势类型只包含一个缠中说禅走势中枢
>注意关键次，*完成的*和*走势类型*

**缠中说禅趋势**：在任何级别的任何走势终，某完成的走势类型至少包含两个以上依次同向的缠中说禅走势中枢
>中枢之间必须绝对不存在任何重叠，一瞬间都不行

**缠中说禅技术分析基本原理1**：任何级别的任何走势类型终要完成
>真传一句话：走势终完美

**缠中说禅技术分析基本原理2**：任何级别任何完成的走势类型，必然包含一个以上的缠中说禅走势中枢
>没有中枢就等中枢出现，原理2不是废话，在主升浪中让你拿住

**缠中说禅走势中枢定理1**：在趋势中，连接两个同级别缠中说禅走势中枢的必然是次级别及次级别以下的走势类型
>注意前提条件：*趋势*
>连接5分钟级别的肯定是1分钟级别的一笔，甚至是几根K线

**缠中说禅走势中枢定理2**：在盘整中，无论是离开还是返回“缠中说禅走势中枢”的走势类型必然是次级别以下的
>这个定理没有完全理解，盘整必然是一笔？如果是这样，那就懂了

**缠中说禅走势中枢定理3**：某级别“缠中说禅走势中枢”的破坏，当且仅当一个次级别走势离开该“缠中说禅走势中枢”后，其后的次级别回抽走势不重新回到该“缠中说禅走势中枢”内

**场中说禅走势分解定理一**：任何级别的走势，都可以分解成*同级别*「盘整」，「下跌」与「上涨」三种走势类型的连接。
>注解：同级别，要求笔跟笔，线段跟线段，不可混为一谈，走势分来分去就这三种，再无其他

**场中说禅走势分解定理二**：任何级别的任何走势类型，都至少由三段以上次级别走势类型构成
>两个关键点
>* 至少三段
>* 次级别走势类型

**缠中说禅走势级别延续定理1**：在更大级别缠中说禅走势中枢产生前，该级别走势类型将延续
>不是很懂
>*我的理解就是如果一个判断不出来的中枢在走的时候，如果没有更大级别的中枢产生，都默认延续

**缠中说禅走势级别延续定理2**：更大级别缠中说禅走势中枢产生，当且仅当围绕连续两个同级别缠中说禅走势中枢产生的波动区间产生重叠
>不是很懂
>对象是：*大级别的缠中说禅走势中枢产生*
>判断方法是：*波动区间产生重叠*
>问题是，要怎么处理呢？进出策略是什么呢？

**缠中说禅走势中枢中心定理1**：走势中枢的*延伸*等价于任意区间[dn,gn]与[ZD，ZG]有重叠
>如果有一根反击，完全出了中枢，且回踩不回去中枢，那说明肯定会有高级别的走势中枢产生或者趋势（中枢新生）

**缠中说禅走势中枢中心定理2**：前后同级别的两个缠中说禅走势中枢，后GG<前DD，等价于下跌及延续；后DD>前高高等价于上涨及其延续


**第三类买卖点定理**：一个次级别走势类型向上离开缠中说禅走势中枢，然后以一个次级别走势类型回试，其低点不跌破ZG，则构成第三类买点；一个次级别走势类型向下离开缠中说禅走势中枢，然后以一个次级别走势类型回抽，其高点不升破ZD，则构成第三类卖点。

#### 中枢的级别到底怎么看？

缠中说禅走势中枢：某级别走势类型中，被至少三个连续次级别走势类型所重叠的部分。

>在30分钟级别的走势中，被至少三个连续5分钟级别走势类型所重叠的部分，也就是5分钟一笔构成5分钟线段，

5分钟线段=5分钟级别的走势类型=30分钟的一笔走势

三个5分钟走势类型重叠的部分，即三个5分钟线段重叠的部分，即30分钟级别的线段中枢

中枢的定义就是第一种，上下上，下上下，上下上下上的是更高级别的中枢

中枢里上下上的就是级别升级

>并非说让你去5分钟k线图上去画笔线段和中枢，然后确定这是5分钟级别的走势类型。

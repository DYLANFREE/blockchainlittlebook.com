关于分型与笔的一些补充性知识点
顶底分型互相包含，比如下跌中一个弱势反弹，虽然有顶分型，但是顶分型被下跌的底分型的大阴柱包含，这种情况下，可以选择成笔，可以选择不成笔，在小级别里一般不成笔，在大级别图里一般是成笔的

大周期可以用新笔定义，因为重点在转折，也就是分型以及是否成笔

## 缠论里的定理

中枢的魔力根源是什么？
缠中说禅：所有人的贪嗔痴

**缠中说禅盘整**：在任何级别的任何走势中，某完成的走势类型只包含一个缠中说禅走势中枢
>注意关键次，*完成的*和*走势类型*

**缠中说禅趋势**：在任何级别的任何走势终，某完成的走势类型至少包含两个以上依次同向的缠中说禅走势中枢
>中枢之间必须绝对不存在任何重叠，一瞬间都不行

**缠中说禅技术分析基本原理1**：任何级别的任何走势类型终要完成
>真传一句话：走势终完美

**缠中说禅技术分析基本原理2**：任何级别任何完成的走势类型，必然包含一个以上的缠中说禅走势中枢
>没有中枢就等中枢出现，原理2不是废话，在主升浪中让你拿住

**缠中说禅走势中枢定理1**：在趋势中，连接两个同级别缠中说禅走势中枢的必然是次级别及次级别以下的走势类型
>注意前提条件：*趋势*
>连接5分钟级别的肯定是1分钟级别的一笔，甚至是几根K线

**缠中说禅走势中枢定理2**：在盘整中，无论是离开还是返回“缠中说禅走势中枢”的走势类型必然是次级别以下的
>这个定理没有完全理解，盘整必然是一笔？如果是这样，那就懂了

**缠中说禅走势中枢定理3**：某级别“缠中说禅走势中枢”的破坏，当且仅当一个次级别走势离开该“缠中说禅走势中枢”后，其后的次级别回抽走势不重新回到该“缠中说禅走势中枢”内

**场中说禅走势分解定理一**：任何级别的走势，都可以分解成*同级别*「盘整」，「下跌」与「上涨」三种走势类型的连接。
>注解：同级别，要求笔跟笔，线段跟线段，不可混为一谈，走势分来分去就这三种，再无其他

**场中说禅走势分解定理二**：任何级别的任何走势类型，都至少由三段以上次级别走势类型构成
>两个关键点
>* 至少三段
>* 次级别走势类型

**缠中说禅走势级别延续定理1**：在更大级别缠中说禅走势中枢产生前，该级别走势类型将延续
>不是很懂
>*我的理解就是如果一个判断不出来的中枢在走的时候，如果没有更大级别的中枢产生，都默认延续

**缠中说禅走势级别延续定理2**：更大级别缠中说禅走势中枢产生，当且仅当围绕连续两个同级别缠中说禅走势中枢产生的波动区间产生重叠
>不是很懂
>对象是：*大级别的缠中说禅走势中枢产生*
>判断方法是：*波动区间产生重叠*
>问题是，要怎么处理呢？进出策略是什么呢？

**缠中说禅走势中枢中心定理1**：走势中枢的*延伸*等价于任意区间[dn,gn]与[ZD，ZG]有重叠
>如果有一根反击，完全出了中枢，且回踩不回去中枢，那说明肯定会有高级别的走势中枢产生或者趋势（中枢新生）

**缠中说禅走势中枢中心定理2**：前后同级别的两个缠中说禅走势中枢，后GG<前DD，等价于下跌及延续；后DD>前高高等价于上涨及其延续


**第三类买卖点定理**：一个次级别走势类型向上离开缠中说禅走势中枢，然后以一个次级别走势类型回试，其低点不跌破ZG，则构成第三类买点；一个次级别走势类型向下离开缠中说禅走势中枢，然后以一个次级别走势类型回抽，其高点不升破ZD，则构成第三类卖点。

#### 第一类买卖点
**缠中说禅买卖点定律1**：任何级别的第二类买卖点都有次级别相应走势的第一类买卖点构成

**缠中说禅趋势转折定律**：任何级别的上涨转折都是由某级别的第一类卖点构成的；任何的下跌转折都是由某级别的第一类买点古城的

#### 第三类买卖点
**第三类买卖点定理**：一个次级别走势类型向上离开缠中说禅走势中枢，然后以一个次级别走势类型回试，其低点不跌破ZG，则构成第三类买点；一个次级别走势类型向下离开缠中说禅走势中枢，然后以一个次级别走势类型回抽，其高点不升破ZD，则构成第三类卖点。

>注意，看的是次级别走势的回抽的低点有无跌破中枢，如果是5分钟级别的操作，介入5分钟级别的笔的第三类买点，那就应该看1分钟的级别的一笔回抽（顶底分型都出现），如果底分型的低点不触及中枢ZG，则可以介入

## 还没搞懂的知识

中枢延伸

>这种情况，只要对中枢延伸的数量进行限制，就可以消除多义性，一般来说，中枢的延伸不能超过5 段，也就是一旦出现6 段的延伸，加上形成中枢本身那三段，就构成更大级别的中枢了

中阴阶段
生生不息，死死不息，在生与死之间的状态就叫做中阴。
在第一类买卖点启动时意味着中阴的开始。
第二类买卖点所处区域位置意味着所处中阴状态。
第三类买卖点的出现意味着中阴阶段的结束。

用Boll来辅助判断中阴阶段结束时间的技巧是：

某级别中阴阶段的结束，一般对应着高一级别Boll轨道收口。

最小级别的中枢是1F级别由3个1分钟线段构成的

在缠论中，因为最小级别的走势是1分级别，那么1F级别的中枢是由三个1F级别的线段构成。

至少三个1分钟的线段重叠部分构成1分钟中枢，至少含有1个1分钟中枢走势为1分钟级别走势，1分钟级别的走势分为趋势和盘整两种走势类型；
>具体到操作上要怎么样，在5分钟图上，画5分钟的笔，5分钟的笔构成5分钟的线段，三个5分钟的线段构成5分钟级别的中枢，5分钟中枢构建5分钟级别走势，分为趋势和盘整，也就是我现在画的黄色中枢才是代表5分钟级别的中枢，把黄色线段当成笔，构建的蓝色线段，3个蓝色线段构成的是30分钟级别的中枢，但是又对不上啊，是本来就不应该对上的是吗，级别与K线图上的时间周期本来就是两回事？

## 今天一定要彻底搞明白的知识

线段到底怎么样才算终结

* 第三笔被第一笔包含，要看是破前低还是前高，最后走出来才能确定情况如何，但结果只有两个是确定的，要么延续，要么结束
* 第三笔才出现笔破坏，即有缺口的情况，但是5点必须高于7点，如果7点高于5点，说明这一笔破坏被更强的一刀捅回来了，所以线在延续，如果第三笔是一笔笔破坏，但是之后无法形成反向分型，被打回来了。

>错误点：把每一个底背驰都当作一买，导致亏损

*血的教训*：不管是一买二买三买，都必须是下跌趋势的完美，也就是有两个同级别的中枢之后，才会出现

正确的做法：底背驰进入，顶背驰出来，波段

**疑**
买点不买，总是在错过后追高，要么不做一买，做的话就在下跌中买入

正确的做法：

1. 仓位控制：1成1买，3成2买，6成三买
2. 只做5分钟级别的操作，如果5分钟和1分钟都出现背驰，才在1买位进行买入
3. 前低挂止盈止损委托

**贪**
卖点不卖，不说了，死太多次
